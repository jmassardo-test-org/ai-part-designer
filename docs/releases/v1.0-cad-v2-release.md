# CAD v2 Release Summary

## v1.0 Release - January 30, 2026

### Overview

The CAD v2 refactor is complete and ready for production release. The new system replaces the legacy v1 pipeline with a declarative schema-based approach using Build123d for geometry generation.

---

## Completed Phases

### Phase 1: Enhanced Geometry Features ✅
- Build123d compiler with hollow enclosure generation
- Snap-fit lid geometry
- Ventilation slots (multiple patterns)
- Feature cutouts (ports, buttons, displays)
- Corner radius support
- Multi-part export (body + lid)

### Phase 2: Frontend API Migration ✅
- v2 API client functions (`lib/generate-v2.ts`)
- React Query hooks (`useGenerateV2.ts`)
- Async generation with Celery background tasks
- Job status polling and WebSocket updates
- Save-to-project functionality

### Phase 3: UI Component Updates ✅
- GeneratePageV2 with AI and Manual modes
- Port/cutout feature selection UI (7 presets)
- ModelViewer loading from v2 download URLs
- STEP + STL dual-format downloads
- History management with delete functionality

### Phase 4: Legacy Cleanup & Documentation ✅
- Lazy loader updated for GeneratePageV2
- API documentation (`docs/architecture/api-v2-cad.md`)
- User guide updated with Enclosure Generator section
- CAD_V2_AS_DEFAULT enabled

### Phase 6: Testing & Validation ✅
- E2E tests for generation flow (18 tests)
- E2E tests for project management (15 tests)
- Backend integration tests (9 tests)
- Security checklist verified
- Accessibility audit completed (WCAG 2.1 AA)

## Key Features

### AI Mode
- Natural language description input
- Example prompts for quick start
- Schema preview before compilation
- Generated schema display

### Manual Mode
- Dimension presets (Small, Medium, Large, Pi Case, Arduino)
- Custom dimensions with sliders
- Wall thickness and corner radius
- Lid type selection (Snap-fit, Screw-on, Hinged)
- Ventilation patterns (Slots, Honeycomb, Circular)
- Port presets (USB-C, USB-A, HDMI, Ethernet, Power Jack, SD Card, Audio)
- Side selection for each port

### Export
- Multi-format export (STEP + STL)
- Multi-part support (body, lid)
- Job-based download URLs
- Background compilation for complex parts

### Project Integration
- Save generated designs to projects
- Design metadata persistence
- Enclosure spec storage for re-editing

## API Endpoints

| Endpoint | Description |
|----------|-------------|
| `POST /api/v2/generate/` | Generate from description |
| `POST /api/v2/generate/preview` | Preview schema only |
| `POST /api/v2/generate/compile` | Compile schema to CAD |
| `GET /api/v2/generate/job/{id}/status` | Job status polling |
| `GET /api/v2/downloads/{job_id}/{filename}` | Download files |
| `POST /api/v2/designs/save` | Save to project |
| `GET /api/v2/designs/` | List saved designs |
| `GET /api/v2/designs/{id}` | Get design details |
